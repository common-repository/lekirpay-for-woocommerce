Integrate your WooCommerce plugin with Lekirpay payment gateway.

== Description ==

Accept payment on your WooCommerce store through multiple payment channels via Lekirpay payment gateway.

Signup Lekirpay merchant account [here](https://app.Lekirpay.com/) for live or [here](https://sandbox.Lekirpay.com/) for sandbox.

== Installation ==

**Step 1:**

- Login to your *WordPress Dashboard*
- Navigate to **Plugins >> Add New**
- Search **Lekirpay for WooCommerce >> Install Now**

**Step 2:**

- Activate Plugin

**Step 3:**

- Navigate to **WooCommerce >> Settings >> Payment >> Lekirpay** and click Manage
- Insert your **Client ID**, **Lekirkey** and **Lekir-Signature Key (optional)**
- Save changes
